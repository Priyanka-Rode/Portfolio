# 🚀 Priyanka Rode Portfolio

🌐 **Live Demo:** https://priyanka-rode-portfolio.netlify.app

---

## 📌 About the Project

This is my personal **developer portfolio website** showcasing my skills, projects, and experience as a **Full Stack Java Developer**.

It highlights my technical expertise in building scalable applications and provides an easy way for recruiters and collaborators to connect with me.

---

## ✨ Features

* 🎨 Modern UI with glassmorphism & gradients
* ⚡ Smooth animations and transitions
* ⌨️ Typing animation in hero section
* 💻 Auto-fetch GitHub repositories using API
* 📩 Contact form integration
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Bootstrap
* GitHub API
* EmailJS

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/Priyanka-Rode/portfolio.git
```

2. Navigate to project folder

```
cd portfolio
```

3. Open `index.html` in browser

---

## 🔑 Configuration

### GitHub API

Make sure your GitHub username is set in `script.js`:

```
https://api.github.com/users/Priyanka-Rode/repos
```
